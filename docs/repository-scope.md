# Repository Scope

This repo is a checkpoint, not a redistributable Office bundle.

Committed:

- Reproduction notes
- Build notes
- Helper scripts
- Candidate source diffs
- Progress log
- Proof screenshots

Not committed:

- Microsoft Office installers or payload caches
- CrossOver `.deb` files or installed CrossOver trees
- Office prefixes or bottles
- Compiled Wine output
- Large downloaded source archives

The target end state is still a fully scripted open Wine/Proton path. The
current checkpoint proves that the locally built CodeWeavers-source Wine can run
Excel when given a known-good Office prefix.
